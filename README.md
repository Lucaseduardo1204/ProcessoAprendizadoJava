# ProcessoAprendizadoJava
Java – Processo de Aprendizado

Repositório dedicado ao meu estudo da linguagem Java e seus principais frameworks, seguindo o curso “Java COMPLETO: Programação Orientada a Objetos + Projetos” do professor Nélio Alves (Udemy).
Aqui organizo conceitos, exercícios, projetos e anotações, com commits que mostram minha evolução ao longo do curso.

# Objetivo do Repositório

- Consolidar meu aprendizado em Java e boas práticas de programação.
- Criar uma trilha de commits mostrando meu progresso para recrutadores.
- Documentar conceitos de forma clara, demonstrando entendimento real do que foi estudado.
- Desenvolver projetos práticos usando tecnologias modernas do ecossistema Java.
